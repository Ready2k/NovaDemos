# Documentation Organization

This document describes the documentation structure for the Voice S2S project.

## Structure

### `/docs/a2a/`
Agent-to-Agent (A2A) multi-agent system documentation:
- A2A system README and implementation details

### `/docs/architecture/`
Architectural documentation and technical guides:
- Native tool solutions
- System architecture decisions

### `/scripts/`
Utility scripts for development and operations:
- `a2a.sh` - A2A system utilities
- `container.sh` - Container management

### Root Level
Essential project files:
- `README.md` - Main project documentation
- `CHANGELOG.md` - Version history
- `LICENSE` - Project license
- Docker compose files for different deployment modes
- Package management files

## Branch Structure

- `main` - Production-ready code
- `industrialisation` - Infrastructure and deployment improvements
- `experimental-cleanup` - Current experimental branch for testing and cleanup
